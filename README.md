# racional-test
Racional Technical Test

## Descripción del Proyecto

Este proyecto es una aplicación full-stack para la gestión de portafolios de inversión, compuesta por:

- **racional-api**: API REST desarrollada con NestJS que proporciona endpoints para gestionar usuarios, transacciones financieras, órdenes de compra/venta de acciones y portafolios de inversión.
- **racional-app**: Aplicación frontend desarrollada con React, TypeScript y Vite que permite visualizar y gestionar portafolios de inversión de forma interactiva.

## Instrucciones de Clonación y Navegación

### Clonar el proyecto

```bash
git clone <url-del-repositorio>
cd racional-test
```

### Navegar entre directorios

Para trabajar con la API:
```bash
cd racional-api
```

Para trabajar con la aplicación frontend:
```bash
cd racional-app
```
